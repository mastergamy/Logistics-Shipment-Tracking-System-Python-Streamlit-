# 📦 Logistics Shipment Tracking System

A simple web-based shipment tracking and management system built using **Python, Streamlit, and Pandas**.  
This project simulates real-world logistics operations such as tracking shipments, updating delivery status, and managing shipment records.

---

## 🚀 Features

- Add new shipment data
- Update existing shipment status
- Prevent duplicate tracking entries
- Prevent updates once shipment is delivered
- View complete tracking history
- Count delivery attempts
- User-friendly interface using Streamlit

---

## 🛠 Tech Stack

- Python
- Streamlit
- Pandas
- CSV (for data storage)

---

## 📂 Project Structure
├── main.py # Main navigation file
├── tracking.py # Tracking page
├── data_update.py # Data entry/update page
├── tracking.csv # Data storage file
├── requirements.txt
├── .gitignore
└── README.md


---

Clone the repository 

Install dependencies
pip install -r requirements.txt

Run the app

streamlit run main.py

This project is inspired by real-world logistics operations and demonstrates how shipment tracking systems work in
